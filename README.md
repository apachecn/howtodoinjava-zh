## 下载

### Docker

```
docker pull apachecn0/howtodoinjava-zh
docker run -tid -p <port>:80 apachecn0/howtodoinjava-zh
# 访问 http://localhost:{port} 查看文档
```

### PYPI

```
pip install howtodoinjava-zh
howtodoinjava-zh <port>
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g howtodoinjava-zh
howtodoinjava-zh <port>
# 访问 http://localhost:{port} 查看文档
```
